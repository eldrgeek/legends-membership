Site Administration

Committee user management and site activity.

Committee Users

Members who have login access to gated committee pages.

Netlify dashboard required for user listing. The Netlify Identity API requires a service-role JWT to list users client-side — that credential is not stored here for security reasons. To view and manage committee users:
Open Identity Dashboard →

The Identity tab shows all registered users, their last login, and confirmation status.

Invite a Committee Member

Send a Netlify Identity invitation so a new member can set a password and access gated pages.

Invitations require the Netlify dashboard. Client-side invite calls require the same service-role JWT that user listing does. Use the button below to go directly to the Netlify invite interface:
Invite User in Netlify Dashboard →

In the Identity tab, click Invite users, enter the email address, and Netlify sends the invite email. The recipient clicks a link to set their password.
